CSS Basics

Introduction to CSS

By now you know how to write in HTML, which sets the structure and format of a webpage. Once you have done this, you now need to tell the browser how to display the information you have given it. You will use CSS to do this! CSS stands Cascading Style Sheets.

CSS is a language that allows you to target elements in your HTML by either, using the generic name of elements or targeting a specific element by giving it a class or ID. Once you have done this you can give them the attributes you desire to style the web pages you are creating.

  p {
      text-height: 2rem;
      text-align: center;
    }

    .card {
      width: 50%;
      height: auto;
      margin: 0 1rem 4rem 0;
      display: inline;
      color: #3fa8cc;
    }

    #second-header {
      padding-top: 2px;
      margin-left: 8px;
      float: right;
      width:auto;
    }

The important thing to remember about CSS is the "Cascading" part of it. This means that if you use the same selector twice, the one that is further down the style sheet will over ride the first one. We call this having more "specificity".

In this example, the background color will be blue since it is at the bottom, but it will keep the width of 100%.

  div {
      width: 100%;
      background: white;
    }

    div {
      background: blue;
    }

Since there is nothing to override the width attribute, it will not be changed unless something more "specific" is written for div.

Syntax

The format you will write CSS in is very easy and repetitive. You first list the element, id, or class that you are going to target, then open up curly braces. Inside the curly braces you list the attribute you wish to add or change with a semi-colon separating it from the value and end with colon. Repeat this for as many attributes are you want, making sure it is all closed up with the correct curly brace. Look at the examples below as a reference. It will also show you how to identify classes and ID's.

    element {
        width: auto;
        height: auto;
        font-size: 18px;
      }

      .class {
        width: 10%;
        border: 10px dotted blue;
      }

      #id {
        display: none;
        background-color: #aacc55;
      }

We have told you how to target classes and ID's in CSS, but we haven't told you have to label elements in HTML with them. You put these labels in the open tag of the element that you wish to target by adding class="" with the name of the class inside the quotes. The same goes for ID's by you write id="" instead. Pretty simple!

  <div class="class-1">
      <h3 id="random-name">I can call them anything!</h3>
      <p>Don't forget hyphens if you use multiple words</p>
    </div>

Just like in HTML you can leave comments in CSS to help keep track of your styles. These won't effect the styles you are writing for your webpage at all but really help keep your code organized for when you have to return to it.

Here is an example:

  /* Header styles! This is a comment!
    You can use them to label sections of styles! */
    .nav {
      width: 100%;
      height: 10rem;
    }

Measurements

There are many different measurements used in CSS. Many of them you will be familiar with, such as, percentages, centimeters, inches, millimeters, and pixels. These are also a few of the most common ones.

Others that you will see quite often are em and rem.
