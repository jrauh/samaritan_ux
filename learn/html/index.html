Introduction to HTML

HTML stands for Hypertext Markup Language. In simplest terms, HTML is used to display the information that either you and/or a program tells it to display.

Here is an example of HTML:

      
        <div class="card small">
          <h3 class="title">
            Title
          </h3>
        </div>
        <div class="great--coding">
          <h2>This is a header tag!<h2>
          <p>This is a paragraph!</p>
        </div>

Elements

HTML is made up of elements. An element is an individual component usually surrounded by similar looking tags. You can determine the type of element and what it contains by looking at the opening and closing tags. A closing tag includes a "/" at the beginning of it..

Beware, not all elements require a closing tag.

      
        <div>
          <h2>This is a header tag!<h2>
          <p>This is a paragraph!</p>
          <input type="text">
        </div>
