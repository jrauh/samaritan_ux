Introduction to HTML

HTML stands for Hypertext Markup Language. In simplest terms, HTML is used to display the information that either you and/or a program tells it to display.

Here is an example of HTML:

  <div class="card small">
      <h3 class="title">
        Title
      </h3>
    </div>
    <div class="great--coding">
      <h2>This is a header tag!<h2>
      <p>This is a paragraph!</p>
    </div>

Ideally you are using the TeamTreehouse course that we recommend to start your learning process. This page is meant to aid you in learning HTML and is not a stand alone resource. Our hope is to help you understand the basics and elaborate on harder to understand aspects.

Elements

HTML is made up of elements. An element is an individual component usually surrounded by similar looking tags. You can determine the type of element and what it contains by looking at the opening and closing tags. A closing tag includes a "/" at the beginning of it.

Beware, not all elements require a closing tag.

  <div>
      <h2>This is a header tag!</h2>
      <p>This is a paragraph!</p>
      <input type="text">
    </div>

Getting Started

There are certain elements every HTML file has to start out with. These are written inside the head element, which gives the computer information, such as: what language you are using, viewport parameters, and where to look for your CSS stylesheets.

Here is an example of what almost every HTML file will start out looking like:

  <!DOCTYPE html>
    <html>
      <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <title>Learn Html</title>
        <link rel="stylesheet" type="text/css" href="../styles/styles.css">
      </head>

      <body>

      </body>
    </html>

It could be helpful to bookmark this page to access the sample above quickly. Copying and pasting it at the beginning of your HTML document is a quick way to get to the enjoyable part of writing in HTML. Plus it's just easier when you don't have to try and remember this while you are learning to code.

Clean Coding

Nesting

Not only can an element contain content of its own but it can also contain other elements and most commonly does. Nesting is accomplished by indenting the lines that are contained inside the element.

An element that contains another element is called a Parent and the element that inside is called a Child. Elements that are on equal levels are called Siblings.

  <div>I'm the Parent Element!!!
      <h2>I'm a child!</h2>
      <p>I'm a child and siblings with the h2 tag above me!</p>
    </div>

Commenting

There is one element in HTML that is not displayed in the browser. This is the comment element. Just like every other element in HTML you use an opening and closing tag to tell the browser that it's a comment and should not be displayed. 

  <!--Example paragraph section of code-->
    <section class="row">
      <h2>Nesting</h2>
      <p>This the paragraph on Nesting. Its long so I made it shorter.</p>
      <p>Paragraph <strong>Parent</strong> Sample called a <strong>Child</strong>. Example <strong>Siblings</strong>.</p>
      <!--example code section of Nesting-->
  <pre class="language-html line-numbers">
    <code class="language-html"><div>I'm the Parent Element!!!
        <h2>I'm a child!</h2>
        <p>I'm a child and siblings with the h2 tag above me!</p>
      </div></code>
  </pre>
    </section>
    <!--Example section ends-->

If you make comments in your HTML as you go, then it will be infinitely easier to find it later. This is super useful as you continue to build and maintain as you build.
